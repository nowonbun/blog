[CentOS] Vim編集ツールをインストールする方法

Devlopment note / Linux    作成日付 : 2019/10/21 00:24:44   修正日付 : 2019/10/21 00:25:09

こんにちは。明月です。


Linuxを使う時、よく使うエディタがVI編集ツールではないかと面ます。でもVI編集ツールはウィンドウOSで考えるとメモ帳ですね。

メモ帳はただのIME編集機能しかないので、普通は「sakura」や「notepad++」を使いますね。


「linux」と同じ状況なので「VI」はただメモ帳なら「VIM」ならターミナルの「sakura」や「notepad++」ですね。


それならインストールをしましょう。

yum list vim*
yum install vim

vi編集ツールとvim編集ツールをhttpd.confファイルでどの差があるかを確認しましょう。
