[CentOS] FTPインストールする方法(vsftpd)

Devlopment note / Linux    作成日付 : 2019/11/01 07:29:57   修正日付 : 2019/11/01 07:30:32

こんにちは。明月です。


前述でSambaのネットワークドライブに関してインストールしました。Sambaの場合は「c class」中で簡単にファイルを電送しますが、問題は「c class」内部ではなく、外部ならSambaを利用ができません。

なので、外部でサーバにファイルを電送するようにはファイルプロトコールが必要ですが、それが「FTP」です。

FTPとは「File Transfer Protocol」の略語でサーバの初期設定でSambaほど重要な設定だと考えられます。


それで「FTP」をインストールしましょう。

yum install vsftpd

インストールは完了しました。設定ファイルで設定しましょう。

vi /etc/vsftpd/vsftpd.conf
#匿名接続禁止
anonymous_enable=NO
#権限設定 644
local_umask=022

サーバ起動時、vsftpdも起動されるような設定です。

systemctl enable vsftpd
systemctl list-unit-files | grep enabled

次はファイアウォールを解除しましょう。

firewall-cmd --permanent --zone=public --add-service=ftp
firewall-cmd --reload
firewall-cmd --zone=public --list-all
systemctl restart vsftpd

次に「SELINUX」を解除する方法です。

vi /etc/sysconfig/selinux
SELINUX=disabled

設定は完了しました。これから接続して確認しましょう。

接続が完了しました。
